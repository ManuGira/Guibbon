Guibbon

The Guibbon object contains the list of open windows

getWindowProperty(prop_id)
not all flags ar handled

cv2.WND_PROP_FULLSCREEN: fullscreen property (can be WINDOW_NORMAL or WINDOW_FULLSCREEN). cv2.WND_PROP_AUTOSIZE: autosize property (can be WINDOW_NORMAL or WINDOW_AUTOSIZE). cv2.WND_PROP_ASPECT_RATIO: window's aspect ration (can be set to WINDOW_FREERATIO or WINDOW_KEEPRATIO). cv2.WND_PROP_OPENGL: opengl support. cv2.WND_PROP_VISIBLE: checks whether the window exists and is visible cv2.WND_PROP_TOPMOST: property to toggle normal window being topmost or not cv2.WND_PROP_VSYNC: enable or disable VSYNC (in OpenGL mode)

createInteractivePolygon(windowName, point_xy_list, label='', on_click=None, on_drag=None, on_release=None, magnet_points=None)

Draws a polygon over the displayed image. The corners of the polygon can be dragged by the user, triggering a callback.

createInteractiveRectangle(windowName, point0_xy, point1_xy, label='', on_click=None, on_drag=None, on_release=None, magnet_points=None)

Create and returns object of type createInteractiveRectangle windows named windowName.

createTrackbar(trackbarName, windowName, value, count, onChange)

Same trackbar as in opencv. However, for a more pythonic signature, you may want to use create_slider(...) instead
